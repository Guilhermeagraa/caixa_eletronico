# Caixa Eletrônico

Projeto simples de simulação de um caixa eletrônico em Python.

## Funcionalidades

- Depositar valores positivos
- Sacar valores respeitando saldo, limite e número máximo de saques
- Visualizar extrato
- Sair do sistema
